# None
파이썬에서 '**값이 없음**'을 표현하는 자료형

## None 표현
```python
variable = None
print(variable)  # None
```
- N은 대문자라는 점 주의!

# Boolean
**참(True)과 거짓(False)**을 표현하는 자료형

## 불리언 표현
```python
bool_1 = True
bool_2 = False

print(bool_1)  # True
print(bool_2)  # False
print(3 > 1)  # True
print('3' != 3)  # True
```
- 비교/논리 연산의 평과 결과로 사용된다.
- 주로 조건/반복문과 함께 사용된다.
- 파이썬에서는 T, F 대문자로 표기!
